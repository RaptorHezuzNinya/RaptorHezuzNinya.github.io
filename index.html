UI/UX assignment

CrowdAboutNow

Assignment: Build a simple single page application! Come up with a nice new design (responsive & mobile first) for the Campaign overview page. Use the OMDB movie api to fetch a collection of movies. Use the data to populate the 'campaign cards' on the overview page. Render 9 'campaign cards' on your new overview page.

Optional: implement search functionality that does a search query to the OMDB api. If the search has a result update the collection on your page with the new search result. Its up to you if you keep a maximum of 9 'campaign cards' or you add the result to the fetched collection.

requirements:

  1. Design a new layout for the campaign overview page
  2. Build a simple SAP with any front-end lib you like (react, lit-element, vue)
  3. Use a http client to fetch a collection of movies from the OMDB API on page load
  4. Render the fetch result on the page in a aesthetically pleasing way
