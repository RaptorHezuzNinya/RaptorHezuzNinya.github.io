UI/UX assignment

CrowdAboutNow

Assignment: Build a simple Single Page Application (SPA)! Come up with an aesthetically pleasing (responsive & mobile first) design for the Campaign overview page. Supply your design as an attachment to this assignment. Use the OMDB movie api to fetch a collection of movies. Use the data from the movie api to populate the "campaign cards" on your newly designed overview page. Render 9 'campaign cards' (movies) on your new overview page.

requirements:

  1. Design a new layout for the campaign overview page
  2. Build a simple SPA with any front-end lib you like (react, lit-element, vue, etc...)
  3. Use a http client to fetch a collection of movies from the OMDB API on page load
  4. Render the fetch result on the page in a aesthetically pleasing way
  5. Supply your design as an attachment to this assignment.

Optional: implement search functionality that does a search query to the OMDB api. If the search has a result, update the collection on your page with the new search result. It's up to you if you keep a maximum of 9 'campaign cards' or you add the result to the fetched collection.
